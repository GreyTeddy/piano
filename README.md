# piano

A simple piano made with p5.js

## the piano can be accessed [here](https://doc.gold.ac.uk/~dntou001/piano/index.html).

## Libraries used from p5.js
- p5.dom.js
- p5.min.js
- p5.sound.js

## The wave can be changed to 
- sine
- triangle
- square
- sawtooth
